# 🎓 Lista de Certificados da Alura

Bem-vindo ao meu projeto de estudante! Este repositório contém uma página simples em **HTML** e **CSS** que lista os certificados que obtive na plataforma **Alura**.

## 📄 Sobre o Projeto

Este projeto tem como objetivo apresentar de forma organizada e estilizada todos os cursos e certificados que concluí na Alura. É uma maneira prática de acompanhar meu progresso e compartilhar minhas conquistas.

## 📝 Funcionalidades

- **Lista de Certificados:** Uma lista detalhada com os nomes dos cursos e links para os certificados.
- **Estilização Responsiva:** Layout adaptável para diferentes dispositivos.

## 🚀 Tecnologias Utilizadas

- **HTML:** Estruturação da página.
- **CSS:** Estilização e responsividade.
